    • Menambang

    Menambang adalah satu-satunya cara mendapatkan bijih untuk mengupgrade peralatan, membuat mesin, dan perhiasan. Di dalam tambang waktu tidak di hitung sehingga ini adalah cara yang baik untuk mendapatkan uang. Jika kamu bermaksud menambang, segera beli keranjang. Jika kamu lelah kamu bisa masuk ke kolam air panas lalu kembali menambang.

    Hasil Tambang Normal:
  • Kantong uang. ( Uang akan langsung masuk ke dompetmu )
  • Junk Ore. ( Batu biasa )
  • Copper Ore. ( Bijih perunggu )
  • Silver Ore. ( Bijih perak )
  • Gold Ore. ( Bijih emas )
  • Mysterile. ( Bijih berwarna biru )

    • Hasil Tambang Musim Dingin:
  • Junk Ore. ( 1 G )
  • Kantong Uang. ( 10 G )
  • Copper Ore. ( 15 G )
  • Silver Ore. ( 20 G )
  • Gold Ore. ( 25 G )
  • Mysterile. ( 40 G )
  • Orichalcum. ( 50 G )
  • Adamantite. ( 50 G )
