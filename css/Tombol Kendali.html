    Tombol Penjelasan
    L1 Bersiul memanggil anjingmu.
    L2 Bersiul memanggil kudamu.
    R1 Menukar peralatan yang sedang di pakai.
    R2 Memasukkan atau mengeluarkan barang dari ranselmu (rucksack).
    Arah D-Pad Menggerakkan karaktermu atau memindahkan menu kursor.
    Segitiga Melihat inventory. Kamu dapat memilih sebuah item di dalam ransel dan menekan tombol segitiga untuk melihat penjelasannya.
    Kotak Menggunakan peralatan yang sedang di bawa.
    Silang Mengambil atau menaruh barang. Kamu dapat menentukan pilihan menu dengan tombol ini.
    Lingkaran Tekan dan tahan tombol ini untuk berlari. Juga digunakan untuk membatalkan pilihan.
    Start Untuk pause dan mengeluarkan layar menu yang berisi informasi tentang banyak hal seperti persentase keberhasilan, data hewan, dan lain-lain.
    Select Tombol untuk melihat peta desa.
